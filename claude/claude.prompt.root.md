# Personal instructions

## Committing

Do not commit unless requested earlier in a session or as part of a project-level `CLAUDE.md` file.

When you do commit, run `git add` and `git commit` as separate commands.

Use single-quoted commit messages with Co-Authored-By attribution.

Example:

```bash
git commit -m 'Do the thing

Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>'
```
